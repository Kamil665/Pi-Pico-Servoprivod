# Управление сервоприводом на Raspberry Pi Pico

## Описание

Пример управления сервоприводом с помощью библиотеки `Servo.h`.

Сервопривод подключается к выводу **GP15** и плавно поворачивается:

- от 0° до 180°;
- затем от 180° до 0°;
- цикл повторяется бесконечно.

## Подключение

| Servo | Raspberry Pi Pico |
|-------|-------------------|
| PWM   | GP15 |
| V+    | VBUS (5V) |
| GND   | GND |

## Код

```cpp
#include <Servo.h>

Servo servo;
int angle;

void setup() {
  servo.attach(15);
  servo.write(0);
}

void loop() {
  for (angle = 0; angle <= 180; angle++) {
    servo.write(angle);
    delay(5);
  }

  for (angle = 180; angle >= 0; angle--) {
    servo.write(angle);
    delay(5);
  }
}
```

## Использование

1. Подключите сервопривод:
   - PWM → GP15
   - V+ → VBUS (5V)
   - GND → GND
2. Загрузите программу.
3. После запуска сервопривод будет непрерывно перемещаться между 0° и 180°.

## Примечание

Обычный сервопривод (например, SG90) **не может вращаться на 360°**. Его диапазон составляет примерно **0–180°**.

Для непрерывного вращения необходим **Continuous Rotation Servo (360°)**. В этом случае управление осуществляется скоростью и направлением, а не углом.
